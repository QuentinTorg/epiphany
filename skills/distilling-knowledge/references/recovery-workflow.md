# Recovery Workflow

Use this workflow when the user asks to recover pending work or when reconciliation resumes after interruption.

1. inspect `memory/views/pending-distillation.md`
2. choose the pending source to resume
3. read the source summary and distillation notes
4. inspect linked topics and action items
5. decide the remaining semantic changes
6. apply the result with `apply_distillation_result.py`

Keep the source pending if contradictions, missing information, or unresolved ambiguity still block completion.
