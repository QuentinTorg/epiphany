# Deep Distillation Checklist

Before applying a deep-distillation result:

1. Read the full source thread or import record.
2. Read directly linked topics and action items.
3. Search typed topic indexes for likely existing matches.
4. Decide whether each affected subject should update an existing topic or create a new one.
5. Decide whether action items should be created, updated, resolved, or left pending.
6. Decide whether the source can be marked `complete` or must remain `pending`.

Leave the source pending when:

- contradictions remain unresolved
- user clarification is still required
- the semantic reconciliation is incomplete

Only close a thread after:

- deep distillation succeeded
- the thread state is `complete`
- the user explicitly asked to close it
