# Query Escalation

Escalate from high-level views into deeper evidence only when needed.

Start with:

- `memory/README.md`
- generated views
- typed topic indexes

Escalate to:

- topic and action-item documents

Escalate further to:

- thread snippets
- import records
- normalized import text

Use the original imported binary only when normalized text is insufficient or the user explicitly needs the original artifact.

If pending distillation affects the likely answer, say so before presenting the result as settled.
