# Citation Rules

Use inline source tags for factual claims in derived prose.

Allowed evidence anchors:

- thread snippet: `thr-YYYY-MM-DD-<slug>#snp-0001`
- normalized import text chunk: `imp-YYYY-MM-DD-<slug>#txt-0001`

Rules:

- cite raw evidence, not only other summaries
- keep citations close to the claim they support
- when evidence is incomplete or pending, say so explicitly
- if `query_memory.py` reports pending-distillation warnings, reflect that uncertainty in the answer

Generated views are navigation aids, not primary evidence.
